# Wallet — приложение для учёта личных финансов

Кроссплатформенное приложение для учёта личных доходов и расходов с
категоризацией операций, визуализацией, финансовыми целями, напоминаниями
о платежах и **шифрованием локальной базы данных**. Данные хранятся только
на устройстве пользователя и не передаются на внешние серверы.

## Возможности

- учёт доходов и расходов с автоматическим пересчётом баланса счёта;
- категоризация операций (предопределённые и пользовательские категории);
- аналитика: расходы по категориям, соотношение доходов и расходов;
- финансовые цели с отслеживанием прогресса;
- напоминания о предстоящих платежах;
- защита данных: шифрование локальной БД, вход по паролю.

## Технологический стек

| Назначение | Технология |
|---|---|
| Язык | Python 3.12 |
| Интерфейс | Kivy (кроссплатформенный GUI) |
| Хранение | SQLite (встраиваемая БД) |
| Шифрование | AES-256-GCM, ключ из пароля через PBKDF2 |
| Уведомления | Plyer |
| Диаграммы | Matplotlib |
| Тесты / качество | pytest, Bandit, pip-audit |

> **О шифровании.** База данных шифруется целиком «в покое» средствами
> библиотеки `pycryptodome` (AES-256-GCM), а ключ выводится из пароля
> пользователя функцией PBKDF2. Библиотека выбрана в том числе из-за простой
> сборки под Android (не требует Rust, в отличие от `cryptography`).
> В качестве производственной альтернативы рассматривается расширение
> **SQLCipher** (прозрачное постраничное шифрование) — алгоритм AES-256 и
> схема вывода ключа при этом сохраняются.

## Архитектура

Многослойная архитектура с разделением ответственности:

```
wallet/
├── models/        # модели предметной области (сущности)
├── core/          # инфраструктура: security (шифрование), db (доступ к SQLite)
├── repositories/  # слой доступа к данным (паттерн Repository)
├── services/      # слой бизнес-логики
├── ui/            # слой представления (Kivy)
└── app_context.py # сборка компонентов (внедрение зависимостей)
```

## Установка и запуск

```bash
pip install -r requirements.txt
python -m wallet.ui.app
```

## Сборка apk
```bash
docker run --rm -v "${PWD}:/home/user/hostcwd" -v "wallet_buildozer:/home/user/.buildozer" kivy/buildozer android debug
```

## Тестирование

```bash
pip install pytest
pytest
```
